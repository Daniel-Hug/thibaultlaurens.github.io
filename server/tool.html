Tool

Tools for software development, productivity, debugging, advising, monitoring and collaboration

.NET and Microsoft environment :

  • dotPeek  is the decompiler you need to see how things work in dlls. You can also find  reflector or JustDecompile.
  • NUnit (for unit testing, you know, this things you should do first of all :), moq (for mocking library) & WatiN (for automated functional tests) are my tools for testing.
  • ANTS Performance profiler & ANTS Memory profiler Profiler, they are probably my favorite, I can check what's wrong with my code (at performance or memory level), at the exact line. I use SQL Server Profiler for database and query profiling and, if you like NHibernate : NHibernate Profiler can quickly become essential for you.
  • For logging I use log4net but you can find NLog as well. There is also a very cool (and well known) tool : elmah for errorr logging : very easy to use with a user friendly UI.
  • Many people say it's for beginners, but trust me, code quality tool will be useful sometimes. I used Resharper from long time and you also have CodeRush.
  • Like profilers, code analysis tool make me a little crazy :), especially Nitriq wich is the best code reviews tool I have ever used and Atomiq which can find duplicate code and dependencies in your projects. You also have  the well known NDepend with its graphs and its treemaps !
  • Another tool pretty useful : if you want to convert, or map, objects... with other objects, it can be very long and just so boring. Use AutoMapper, it will make your life easier.
  • If you want to easily define validation rules on your business objects : Fluent Validation is the tool you need.
  • "Kiss goodbye to SQL Management Studio!" it's from LINQPad : an interactive tool to query your databases.
  • Quartz.NET & Castle scheduler are components to schedule jobs in your .net applications.
  • On the front end side, I have some tool for loading and performance optimization like nQuant which can reduce the image size 3 times smaller, SquishIt for bundling your css and javascript and RequestReduce which can do alone what the previous can.
  • Azure Storage Explorer is just essential on azure projects. It allows you to view contents of your blobs, queues and table.
  • If you are a Windows phone developer, you will probably like Windows phone power tools to install and update .xap files on your devise and to browse your IsolatedStorage. I also have a good thing : the one page Windows Phone 7 App Site template or how to quickly put a page online to promote your app. Finally, very useless but just for fun : Windows Phone 7 Emulator Skin Switcher : switch the skin of your Windows Phone 7 emulator !
  • Visual Studio Add-ins, the list is pretty long, here are my favorites : Productivity Power Tools, tangible T4 editor, ankhSVN and Git Source Control Provider
That's it for now, I will complete this list later !
